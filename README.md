\# VLSI AI Design Analyzer



End-to-end AI-powered platform for VLSI design analysis.



\## Features

\- RTL-based PPA prediction (Power, Performance, Area)

\- Layout hotspot detection

\- React frontend + FastAPI backend

\- MongoDB integration

\- Cloud-ready (Azure ML compatible)



\## Tech Stack

\- Frontend: React

\- Backend: FastAPI

\- Database: MongoDB

\- ML: PyTorch / CNN / MLP



\## Status

Core system completed. Cloud deployment planned.
